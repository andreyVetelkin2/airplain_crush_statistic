# Инструкция по запуску системы

## Быстрый запуск

### 1. Установка зависимостей (если еще не установлено)

```bash
pip install -r requirements.txt
```

### 2. Запуск сервера

#### Windows (PowerShell):
```powershell
python backend/main.py
```

#### Windows (CMD):
```cmd
python backend\main.py
```

#### Linux/Mac:
```bash
python backend/main.py
```

### 3. Открытие в браузере

После запуска сервера откройте в браузере:
```
http://localhost:8000
```

## Использование системы

### Основные функции:

1. **Заполнить случайными значениями**
   - Нажмите кнопку "Заполнить случайными значениями"
   - Введите вероятность (0-1) для генерации коэффициентов
   - Все поля будут заполнены случайными данными

2. **Сохранить значения**
   - Нажмите "Сохранить значения"
   - Файл `aviation_safety_data.json` будет скачан

3. **Загрузить значения**
   - Нажмите "Загрузить значения"
   - Выберите ранее сохраненный JSON файл

4. **Вычислить результат**
   - Нажмите "Вычислить результат" (большая зеленая кнопка)
   - Система выполнит расчет и отобразит 2 графика:
     - График 1: Динамика всех переменных системы (X1-X8)
     - График 2: Прогноз количества катастроф (X2)
   - Результаты автоматически сохраняются в localStorage браузера

### Структура данных:

**Таблица 1: Исследуемые показатели (X1-X8)**
- X1 - Нарушения инструкций пилотами
- X2 - Количество катастроф
- X3 - Коэффициент повторяемости происшествий
- X4 - Доля частных судов
- X5 - Метеорологические службы
- X6 - Контроль контрафакта
- X7 - Возраст воздушных судов
- X8 - Квалификация персонала

**Таблица 2: Возмущения (F1-F5)**
- Полиномы 3-й степени: F_i = a·t³ + b·t² + c·t + d

**Таблица 3: Уравнения связей**
- 20 уравнений взаимосвязей между переменными

## Технические детали

- **Бэкенд**: FastAPI на Python
- **Фронтенд**: Bootstrap 5, Vanilla JavaScript
- **Математическая модель**: Метод Рунге-Кутта 4-го порядка
- **Графики**: Matplotlib (генерация на сервере, отображение в base64)

## Решение проблем

### Сервер не запускается:
```bash
# Проверьте установлены ли все зависимости
pip install -r requirements.txt

# Проверьте занят ли порт 8000
netstat -ano | findstr :8000  # Windows
lsof -i :8000  # Linux/Mac
```

### Графики не отображаются:
- Проверьте консоль браузера (F12) на наличие ошибок
- Убедитесь что сервер запущен и доступен
- Проверьте что все поля заполнены числами

### Проблемы с сохранением/загрузкой:
- Убедитесь что разрешены скачивания в браузере
- Проверьте формат JSON файла (должен содержать все необходимые поля)
